# Análise e Previsão de Preço de Casas

## Objetivo
Criar um modelo simples de Machine Learning para prever o preço de casas com base no tamanho.

## Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Etapas do projeto
- Leitura de dados CSV
- Análise exploratória
- Visualização de dados
- Criação de pipeline
- Treinamento de modelo
- Previsão de preços

## Resultado
O modelo conseguiu identificar a relação entre tamanho da casa e preço.
